There are two methods to installing this application:

### Method 1 (easiest)
1. Download the latest release found at [releases](https://github.com/WSU-4110/MOST/releases/edit/untagged-2de59f3c4dfb7414d3c7).
2. Run the installer and save the application to the desired location of install.
3. Run the application

### Method 2
1. Download the latest zip file found at [releases](https://github.com/WSU-4110/MOST/releases/edit/untagged-2de59f3c4dfb7414d3c7).
2. Unzip the contents to a desired location
3. Run the MOST.exe found inside the unzipped file
