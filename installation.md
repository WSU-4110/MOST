## Minimum System Requirements
* OS: Windows 11 (64-bit)
* CPU: Any x64-compatible processor
* Memory: 256MB RAM
* Disk Space: 300 MB available
* Graphics: Integrated graphics

## Recommended System Requirements
* OS: Windows 11
* CPU: Intel Core i3 or better
* Memory: 4GB RAM
* Disk Space: 500 MB available
* Graphics: Integrated graphics or dedicated GPU
* Other: SSD for faster load times

## Install
There are two methods to installing this application:

### Method 1 (easiest)
1. Download the latest release found at [releases](https://github.com/WSU-4110/MOST/releases/tag/v1.0).
2. Run the installer and save the application to the desired location of install.<br>
*Note*: If a windows message appears stating an unrecognized app, click "More info" and then "Run anyway".
3. Run the application

### Method 2
1. Download the latest zip file found at [releases](https://github.com/WSU-4110/MOST/releases/tag/v1.0).
2. Unzip the contents to a desired location
3. Run the MOST.exe found inside the unzipped file
